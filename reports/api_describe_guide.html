API 1: Search aircraft by Location

User need to enter x_loc, y_loc, and image_id of picture.
System will tell user in the location has aircraft, or not.

If system find the aircraft in the location, it will tell user the whole coordinate of the aircraft.
If system didn't find the aircraft in the location, it will tell user not found and let user try again.

Example 1:

Correct_Input Output
x_loc = 1600 Xmin = 1555
y_loc = 500 Ymin = 479
image_id = 5c9e817a-dc4b-42ab-952c-3128e2de12e8.jpg Xmax = 1758
Ymax = 651

Example 2:

Incorrect_Input Output
x_loc = 30000000 "Input X and Y should within (0,2560)"
y_loc = 500
image_id = 5c9e817a-dc4b-42ab-952c-3128e2de12e8.jpg

Example 3:

Incorrect_Input Output
x_loc = 1600 "No image found related to the image_id. Try effective image_id"
y_loc = 500
image_id = wrong_id

API 2: Get all airplanes coordinate in picture

User need to enter an image_id of picture

If system find the picture, it will tell user all coordinates of aircraft in the picture.
If system didn't find the picture, it will tell user not found and let user try again.

Example 1:

Input Output
image_id = 5c9e817a-dc4b-42ab-952c-3128e2de12e8.jpg Xmin = 1950 Xmin = 1990 Xmin = 1989 ...
Ymin = 280 Ymin = 472 Ymin = 722 ...
Xmax = 2139 Xmax = 2182 Xmax = 2187 ...
Ymax = 436 Ymax = 685 Ymax = 894 ...

Example 2:

Incorrect_Input Output
image_id = wrong_id "No image found related to the image_id. Try effective image_id"

API 3: Display the top big or small aircraft in one picture

User need to choose big or small, then enter an image_id of picture and how many aircraft needed.

If system find the picture, it will tell user the top numbers biggest/smallest aircraft in the picture.
If system didn't find the picture, it will tell user not found and let user try again.

Example 1:

Input Output
image_id = 5c9e817a-dc4b-42ab-952c-3128e2de12e8.jpg Xmin = 2067 Xmin = 979 Xmin = 2079
limit_of_number = 3 Ymin = 1344 Ymin = 1641 Ymin = 1574
choose big or small = Big Xmax = 2269 Xmax = 1192 Xmax = 2287
Ymax = 1568 Ymax = 1853 Ymax = 1789

Example 2:

Incorrect_Input Output
image_id = wrong_id "No image found related to the image_id. Try effective image_id"

API 4: Count airplanes in picture

User need to enter an image_id of picture.

If system find the picture, it will tell user how many airplanes in the picture.
If system didn't find the picture, it will tell user not found and let user try again.

Example 1:

Input Output
image_id = 5c9e817a-dc4b-42ab-952c-3128e2de12e8.jpg number of airplanes = 30

Example 2:

Incorrect_Input Output
image_id = wrong_id "No image found related to the image_id. Try effective image_id"

API 5: Search for images by numbers of airplanes

User need to enter two number, one is how many airplanes they want to find, another one is how many pictures they want.

If system find pictures, it will give user those picture system found.
If system didn't find pictures, it will tell user not found and let user try again.

Example:

Input Output
contain_aircraft_number = 32 image_id = 014de911-7810-4f7d-8967-3e5402209f4a.jpg
limit_of_image = 5 image_id = 29324dbf-6043-4521-bd83-44a97ffc5281.jpg
image_id = 70332141-fcd3-4e2e-89c7-d50cbc30c25f.jpg
